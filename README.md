# JsonSchemaScope

Json Schema Scope is a lightweight visual tool that turns JSON data into a clear structural graph. It focuses on field names, value types, optional properties, nested objects, and arrays without exposing the actual data values, making it easier to understand, document, and review complex JSON structures at a glance.

## Features

- Paste JSON directly or import a `.json` file.
- Visualize only the structure: field names, types, nested objects, and arrays.
- Display optional fields with a clear marker.
- Show mixed types such as `string | null` or `array<int | float>`.
- Create graph links only for nested objects and arrays of objects.
- Pan, zoom, recenter, and manually move graph nodes.
- Show JSON parsing errors with line and column information.

## Usage

Open `index.html` in a browser.

No build step or external dependency is required.
